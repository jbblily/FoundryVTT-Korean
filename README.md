##사용법
상단의 링크를 모듈 파인더에 붙여넣어 다운로드하세요. 

모듈 목록에서 (alt) Translation : Korean [Core] 가 다운로드되었는지 확인하세요.

셋업 화면에서 환경설정 > 기본 언어를 (alt) Translation : Korean [Core] 로 변경한 후, 새로고침하세요.

별도로 월드 내에서 모듈을 활성화할 필요는 없습니다.
